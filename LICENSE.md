## 🔧 Common Issues and Fixes

**❌ GUI doesn't appear after execution**
→ Your executor may lack sufficient UNC support. Switch to a Level 7
  executor such as Synapse Z, Delta, or Xeno and re-execute. Ensure
  you are fully loaded into a live game session, not the lobby.

**❌ Inf Money resets after a few minutes**
→ This was patched in v3.1.2. Update to the latest version. If the
  issue persists, toggle Inf Money off and back on using the Economy
  tab to reset the injection loop.

**❌ Auto Attack stops engaging enemies**
→ The targeted enemy territory may have been captured or the enemy
  player left the session. Open the Combat tab, disable Auto Attack,
  wait 5 seconds, then re-enable it to reset targeting.

**❌ Super Spawn generates units but they disappear immediately**
→ You may have hit the server-side troop cap. Use the Auto Upgrade
  Cities feature first to raise your city tiers, which increases
  the maximum troop capacity before running Super Spawn.

**❌ Script works on PC but not on Android**
→ Download the dedicated mobile script version from the link above.
  Some API calls in the PC version are incompatible with mobile
  executor environments.

**❌ Live Stats shows zeros or doesn't refresh**
→ Toggle the Stats tab auto-refresh off and back on. If using Fast
  mode, switch to Slow mode — Fast mode can conflict with server
  rate limiting on low-population servers.

**❌ "attempt to index nil" error on execution**
→ You executed too early. Wait until you have fully selected a country
  and are active in a live Control Europe session before running
  the script.
